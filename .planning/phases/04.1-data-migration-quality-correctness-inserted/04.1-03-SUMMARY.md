# Phase 04.1 Plan 03: Kiosk Table Default Columns Summary

**Kiosk table updated to show Asset (hardware serial number) as first column with 7 default columns per MIGR-12: Asset, Outlet Code, Venue, Region, Stage, CMS Config, Install Date**

- **Duration:** ~8 min

## Accomplishments

- Added `hardwareSerialNumber` to `KioskListItem` type, `listKiosks` select query, and return mapping
- Inserted Asset column as first data column in `kiosk-columns.tsx` with a clickable link to the kiosk detail page
- Moved `kioskId` column to hidden-by-default position (still available via column visibility toggle)
- Updated `kioskDefaultColumnVisibility` to hide `kioskId`, `hardwareModel`, `softwareVersion`, `maintenanceFee`, `freeTrialStatus`, `createdAt`

## Files Created/Modified

- `src/app/(app)/kiosks/actions.ts` — Added `hardwareSerialNumber` to `KioskListItem` type, `listKiosks` select, and return mapping
- `src/components/kiosks/kiosk-columns.tsx` — Added Asset column as position 2 (after select checkbox), moved kioskId to hidden-by-default, updated `kioskDefaultColumnVisibility`

## Decisions Made

- Asset column is a clickable `Link` to `/kiosks/${row.original.id}` — without this, hiding the kioskId column (which was the only navigation link) would leave users with no way to navigate to kiosk detail pages from the table
- kioskId column remains in the column list but hidden by default; operators who prefer it can re-enable it via the column visibility toggle

## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 2 - Missing Critical] Asset column made a clickable detail-page link**
- **Found during:** Task 2 (column definitions)
- **Issue:** The plan specified hiding `kioskId` by default without noting that kioskId was the only navigation link column. Removing the only navigation link would break table usability — users would have no way to open a kiosk detail page from the table view.
- **Fix:** Rendered the Asset cell as a `Link` to `/kiosks/${row.original.id}`, matching the same pattern as the original kioskId column. kioskId was moved to hidden-by-default rather than removed entirely.
- **Files modified:** `src/components/kiosks/kiosk-columns.tsx`
- **Verification:** TypeScript compiles cleanly; Asset cell uses same Link pattern as original kioskId
- **Committed in:** `474aa29` (Task 2 commit)
