    **1. Create `src/components/table/editable-cell.tsx`:**

    A cell component that shows static text by default. On click, it switches to an Input for editing. On blur or Enter, it saves via a `meta.updateField` callback on the table instance.

    ```tsx
    interface EditableCellProps {
      value: string | number | null;
      rowId: string;         // row.original.id
      columnId: string;      // column.id (matches DB field name)
      table: Table<any>;     // to access meta.updateField
      type?: "text" | "number";  // input type
      placeholder?: string;
    }
    ```

    Behavior:
    - Default state: renders the value as text (or placeholder dash if null). Click on the cell stops propagation (`e.stopPropagation()`) and enters edit mode.
    - Edit state: shows an Input with the current value, auto-focused. Styling: no visible border initially, subtle border on focus, matches table cell size (h-8 text-sm).
    - On blur or Enter: call `table.options.meta.updateField(rowId, columnId, newValue)` if value changed. Exit edit mode.
    - On Escape: revert to original value, exit edit mode.
    - The cell click must call `e.stopPropagation()` to prevent row onClick from navigating.

    **2. Create `src/components/table/column-header-filter.tsx`:**

    A column header component that combines the column label, sort indicator, and an inline filter input.

    ```tsx
    interface ColumnHeaderFilterProps {
      column: Column<any, unknown>;
      label: string;
    }
    ```

    Layout:
    - Top row: label text + sort arrow (asc/desc/none). Clicking the label toggles sorting (use `column.getToggleSortingHandler()`).
    - Below label: a small text Input for per-column filtering. Only visible if `column.getCanFilter()` is true.
    - Filter input calls `column.setFilterValue(value)` with debounce (300ms). Use a local state + useEffect debounce pattern.
    - Style the header to be slightly taller to accommodate the filter input — add `py-1` padding.
    - Sort indicators: show ChevronUp for asc, ChevronDown for desc, ChevronsUpDown (faded) for unsorted. Use lucide icons.
    - The filter input should be compact: `h-6 text-xs` with subtle border.
  </action>
