## What Was Built

**Task 1 — actions.ts:**
- `LocationListItem` type extended from 9 fields to 17 (14 data + id + createdAt + archivedAt)

- Maintenance Fee column: `£` prefix applied at render time
- `locationGroupableColumns` extended with sourcedBy, status, region, locationGroup
- `locationFilterableColumns` extended with status, region
