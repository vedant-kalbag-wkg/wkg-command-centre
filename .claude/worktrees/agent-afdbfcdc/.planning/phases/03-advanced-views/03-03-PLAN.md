2. Define `GanttTask` type matching @svar-ui/react-gantt task shape:
   ```typescript
   export interface GanttTask {
     id: string;
     text: string;
     start?: Date;
     end?: Date;
     duration?: number;
     type: "task" | "milestone" | "summary";
     parent?: string;
     open?: boolean;
     color?: string;
     // Custom data for resource column
     members?: { userId: string; userName: string; role: string }[];
   }
   ```

3. `buildGanttTasks(installations, groupBy: "region" | "status")` function:
   - Groups installations by the groupBy field value
   - For each group: creates a `type: "summary"` row with `text: "{group name} . {N} installation{s}"`, `open: true`
   - For each installation in the group: creates a `type: "task"` row with `parent: groupId`, `start: new Date(inst.plannedStart)`, `end: new Date(inst.plannedEnd)`, `color: REGION_COLORS[inst.region] ?? REGION_COLORS.default`, `members: inst.members`
   - For each milestone on the installation: creates a `type: "milestone"` row with `parent: instId`, `start: new Date(ms.targetDate)`, `color: REGION_COLORS[inst.region] ?? REGION_COLORS.default`
   - Installations without plannedStart or plannedEnd are excluded from Gantt (they have no timeline)

**C. Create `src/lib/stores/gantt-store.ts`** — Zustand store for pending drag state:

```typescript
import { create } from "zustand";

interface PendingChange {
  taskId: string;
  originalStart: Date;
  originalEnd: Date;
  newStart: Date;
  newEnd: Date;
  duration: number;
}

interface GanttState {
  pendingChange: PendingChange | null;
  groupBy: "region" | "status";
  zoom: "day" | "week" | "month";
  setPendingChange: (change: PendingChange | null) => void;
  setGroupBy: (groupBy: "region" | "status") => void;
  setZoom: (zoom: "day" | "week" | "month") => void;
  clearPending: () => void;
}

export const useGanttStore = create<GanttState>((set) => ({
  pendingChange: null,
  groupBy: "region",
  zoom: "month",
  setPendingChange: (pendingChange) => set({ pendingChange }),
  setGroupBy: (groupBy) => set({ groupBy }),
  setZoom: (zoom) => set({ zoom }),
  clearPending: () => set({ pendingChange: null }),
}));
```

**Pending-drag intercept pattern** (CRITICAL — from RESEARCH.md Pattern 1):
```typescript
const apiRef = useRef<any>(null);
const init = useCallback((api: any) => {
  apiRef.current = api;
  api.intercept("update-task", (ev: any) => {
    if (ev.inProgress) return; // allow visual tracking during drag
    // Final drop — capture pending state, block library save
    const originalTask = installations.find(i => i.id === ev.id);
    if (originalTask && ev.task?.start && ev.task?.end) {
      setPendingChange({
        taskId: ev.id,
        originalStart: new Date(originalTask.plannedStart),
        originalEnd: new Date(originalTask.plannedEnd),
        newStart: ev.task.start,
        newEnd: ev.task.end,
        duration: ev.task.duration,
      });
    }
    return false; // block library from auto-saving
  });
}, [installations, setPendingChange]);
```
